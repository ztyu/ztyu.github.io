hexo

下载node
npm install -g hexo
hexo init
进入blog目录
npm install hexo-server –save
在git中 部署pubilic 文件到新仓库 每次部署删除后重新关联
hexo g
hexo s


选择主题
进入根目录 npm install hexo-renderer-scss –save
将主题下载到 /themes 目录下：
git clone https://github.com/ahonn/hexo-theme-even themes/even
修改博客根目录下的配置文件（_config.yml），启用 Even 主题。 修改 theme 字段为 even:
Extensions
Plugins: http://hexo.io/plugins/
Themes: http://hexo.io/themes/
theme: even
启动后访问 localhost:4000 预览主题。
创建文章
创建about 配置config打开about 后hexo new page about
hexo new “hexo”
